Mon projet — Vitrine professionnelle de Merph-dev

Présentation du projet : suivi pas-à-pas du parcours autodidacte en développement web, cybersécurité, data engineering et IA, avec suivi Git et démonstrations techniques.

Résumé

Ce dépôt contient le portfolio professionnel de Merph-dev. L'objectif est de documenter et montrer, étape par étape, les compétences et projets dans plusieurs domaines : développement web, cybersécurité, data engineering et IA.

Sections principales du projet

Développement web

  • Pages et composants HTML/CSS responsives
  • Exemples de projets (front-end, back-end, APIs)
  • Démonstrations de bonnes pratiques et optimisation

Cybersécurité

  • Analyses et notes sur tests d'intrusion (pentests) éducatifs
  • Outils, rapports et remédiations
  • Bonnes pratiques de sécurité pour les applications web

Data engineering

  • Ingestion, transformation et pipeline de données
  • Exemples d'ETL, stockage et visualisation
  • Automatisation et orchestration basique

Intelligence artificielle

  • Prototypes et expérimentations en IA/ML
  • Projets miniatures pour illustrer concepts
  • Documentation pédagogique et résultats

Méthodologie & suivi Git

Chaque étape du projet est enregistrée dans l'historique Git pour montrer la progression : commits fréquents, branches pour les fonctionnalités et pull requests pour les intégrations.

Exemple de bonnes pratiques utilisées :

git checkout -b feat/ajout-page-projet
git add .
git commit -m "Ajout de la page explicative du projet"
git push origin feat/ajout-page-projet

Comment parcourir le dépôt

  1. Consulter le README.md (page d'accueil du dépôt) pour le résumé et le plan.
  2. Parcourir les dossiers par technologie (ex: web/, cybersecurity/, data/, ai/).
  3. Regarder les commits et les branches pour suivre la progression pas à pas.

Si vous souhaitez une visite guidée ou une page supplémentaire (ex: page de projets détaillés), dites-moi laquelle et je l'ajoute.
